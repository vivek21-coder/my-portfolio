# Portfolio Website - Vivek Aggarwal

A personal portfolio website showcasing my experience as a Software Engineer and Data Scientist. Built with modern web
technologies to be fast, responsive, and interactive.

## Tech Stack

* **Frontend:** React (Vite)
* **Styling:** Tailwind CSS
* **Icons:** Lucide React
* **Deployment:** GitHub Pages

## How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/YOUR_USERNAME/my-portfolio.git](https://github.com/YOUR_USERNAME/my-portfolio.git)
   cd my-portfolio
   ```

2. **Install Dependencies:**
   ```bash
   npm install
    ```   

3. **Start the development server:**
   ```bash
   npm run dev
   ```

   Open `http://localhost:5173` in your browser
